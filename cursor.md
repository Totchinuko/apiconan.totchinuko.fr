# Change Enhanced Cursor
If you are running Sudo, you'll find in the client's settings a toggle to enable "Normal" cursor. Now I'll save you the explanation as to why I couldn't pak a cursor in the mod, but you'll need to install it your own game. It's just a png to copy somewhere.

[Download Original.png](/API/Original.png ':ignore')

Put the file there:  
`steam\steamapps\common\Conan Exiles\ConanSandbox\Content\UI\Cursors`  
Make sure it is named `Original.png`, and you'll likely need to create the folders `UI` and `Cursors` (watch out for that S on Cursor**s**).

Once done, you should be able to toggle on the setting in Sudo. You can technically put your own PNG cursor in there, with the same name and folder. But be sure to use one that has the pointing end in the top left corner, I've set up the click spot on the corner. This is also using Hardware cursor, so their shouldn't be any concerns about software lag.
